# EcoPonto API

API REST desenvolvida em Node.js para consulta e gerenciamento
de pontos de coleta sustentável.

## Objetivo

Facilitar o acesso da comunidade a informações sobre descarte
correto de resíduos especiais.

## Tecnologias

- Node.js
- Express
- SQLite
- Swagger

## Funcionalidades

- Listar pontos de coleta
- Buscar por material
- Cadastrar pontos
- Atualizar informações
- Excluir registros

## Endpoints

GET /ecopontos
POST /ecopontos
PUT /ecopontos/:id
DELETE /ecopontos/:id

## Projeto Extensionista

Desenvolvido como atividade extensionista voltada à conscientização
ambiental e sustentabilidade.
